Sendungsverfolgung

Sobald Ihre Bestellung versandt wurde, erhalten Sie eine E-Mail mit:

  • Sendungsnummer (Tracking-ID)
  • Link zur Sendungsverfolgung
  • Voraussichtlichem Lieferdatum

Sie können den Status Ihrer Sendung jederzeit online verfolgen.

Versandarten

Standardversand

  • Zustellung an die angegebene Lieferadresse
  • Sendungsverfolgung inklusive
  • Zustellung erfolgt werktags

Paketdienst-Optionen

Je nach Versanddienstleister stehen möglicherweise folgende Optionen zur Verfügung:

  • Wunschnachbar
  • Wunschablageort
  • Packstation (bei DHL)
  • Umleitung an eine andere Adresse

Nicht erreichbar / Nicht zustellbar

Sollten Sie bei Zustellversuch nicht erreichbar sein:

  1. Der Paketdienst hinterlässt eine Benachrichtigungskarte
  2. Das Paket wird in eine Filiale gebracht oder am nächsten Tag erneut zugestellt
  3. Die Lagerzeit beträgt in der Regel 7 Werktage

Bitte sorgen Sie für eine erreichbare Lieferadresse oder nutzen Sie alternative Zustelloptionen (Packstation, Nachbar).

Sendungsstatus

Die Status Ihrer Bestellung können Sie jederzeit einsehen:

  • Im Kundenkonto unter „Bestellungen“
  • In der Versandbestätigungs-E-Mail
  • Direkt beim Versanddienstleister mit Ihrer Tracking-ID
